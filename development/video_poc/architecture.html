What changes at production scale (deliberately not built yet)

Results stop coming from the processor routing

Completed batch artifacts upload to object storage (job doc carries URLs). Live events publish to the platform (vision-events ingest exists today). Nobody ever connects to a worker — same outbound-only rule as the connector.

Live annotated preview moves onto the relay pixels

The processor publishes its annotated output into MediaMTX as just another stream; browsers watch it over WHEP like any source. Encoded only while someone is watching — MJPEG retires.

One warm worker becomes a pool in a cell scale

Relay + worker pool deploy together as a cell (Crusoe/L40S); a job's results stream and WHEP URL point at its cell. Raw video never crosses a cloud boundary. See the video strategy deck for the full argument.
