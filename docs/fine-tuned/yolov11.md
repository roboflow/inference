[YOLOv11](https://yolov11.com), released in September 2024, is a real-time object detection model developed by Ultralytics.

## Supported Model Types

You can deploy the following YOLOv11 model types with Inference:

- Object Detection
- Image Segmentation
- Keypoint Detection
